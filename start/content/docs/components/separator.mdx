    ## Installation

    ```npm
    npx shadcn@latest add @dotui/separator
    ```

    ## Usage

    Use separators to visually divide content in lists, menus, or other places.

    ```tsx
    import { Separator } from "@/components/ui/separator";
    ```

    ```tsx
    <Separator />
    ```

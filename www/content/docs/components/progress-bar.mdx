    ## Installation

    ```npm
    npx shadcn@latest add @dotui/progress-bar
    ```

    ## Usage

    Use progress bars to show the progression of a system operation or task.

    ```tsx
    import { ProgressBar, ProgressBarControl } from "@/components/ui/progress-bar";
    ```

    ```tsx
    <ProgressBar value={60}>
      <ProgressBarControl />
    </ProgressBar>
    ```

    ## Playground

    Use the controls below to experiment with different progress bar props and see the live preview and code update.

    <InteractiveDemo component={ProgressBarPlayground} controls={progressBarControls} />

    ## API Reference

    | Prop              | Type                        | Default | Description                                          |
    | ----------------- | --------------------------- | ------- | ---------------------------------------------------- |
    | `value`           | `number`                    | `0`     | The current value (controlled).                      |
    | `minValue`        | `number`                    | `0`     | The smallest value allowed.                          |
    | `maxValue`        | `number`                    | `100`   | The largest value allowed.                           |
    | `isIndeterminate` | `boolean`                   | `false` | Whether the progress bar is indeterminate.           |
    | `formatOptions`   | `Intl.NumberFormatOptions`  | -       | The display format of the value label.               |
    | `children`        | `ReactNode`                 | -       | The children of the component.                       |

    | Data attribute       | Description                                   |
    | -------------------- | --------------------------------------------- |
    | `[data-indeterminate]` | Whether the progress bar is indeterminate.  |

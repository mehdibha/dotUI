    ## Installation

    ```npm
    npx shadcn@latest add @dotui/badge
    ```

    ## Usage

    Use badges to highlight status, categorize items, or show counts.

    ```tsx
    import { Badge } from "@/components/ui/badge";
    ```

    ```tsx
    <Badge>Badge</Badge>
    ```

    ## Playground

    Use the controls below to experiment with different badge props and see the live preview and code update.

    <InteractiveDemo component={BadgePlayground} controls={badgeControls} />

    ## API Reference

    | Prop      | Type                                                            | Default     | Description                   |
    | --------- | --------------------------------------------------------------- | ----------- | ----------------------------- |
    | `variant` | `'default' \| 'success' \| 'warning' \| 'danger' \| 'info' \| 'outline'` | `'default'` | The visual style of the badge. |
    | `size`    | `'sm' \| 'md' \| 'lg'`                                          | `'md'`      | The size of the badge.        |
    | `children`| `ReactNode`                                                     | -           | The content of the badge.     |

## Installation

```npm
npx shadcn@latest add @dotui/markdown-editor
```

## Usage

Compose the editor from its parts. The `value` is fully controlled — the
toolbar actions and the Write panel both call `onChange` with the next source.

```tsx
import {
  MarkdownEditor,
  MarkdownEditorPreview,
  MarkdownEditorToolbar,
  MarkdownEditorWrite,
} from '@/components/ui/markdown-editor'
```

```tsx
const [value, setValue] = React.useState('# Hello')

<MarkdownEditor value={value} onChange={setValue}>
  <MarkdownEditorToolbar />
  <MarkdownEditorWrite />
  <MarkdownEditorPreview />
</MarkdownEditor>
```

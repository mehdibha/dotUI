    ## Installation

    ```npm
    npx shadcn@latest add @dotui/accordion
    ```

    ## Usage

    Accordions organize content into collapsible sections, allowing users to expand and collapse content as needed. They are useful for reducing visual clutter and letting users focus on relevant information.

    ```tsx
    import {
      Accordion,
      AccordionHeading,
      AccordionItem,
      AccordionPanel,
    } from "@/components/ui/accordion";
    ```

    ```tsx
    <Accordion>
      <AccordionItem>
        <AccordionHeading>Accordion Heading</AccordionHeading>
        <AccordionPanel>Accordion Panel</AccordionPanel>
      </AccordionItem>
      <AccordionItem>
        <AccordionHeading>Accordion Heading</AccordionHeading>
        <AccordionPanel>Accordion Panel</AccordionPanel>
      </AccordionItem>
    </Accordion>
    ```

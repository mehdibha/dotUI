    ## Installation

    ```npm
    npx shadcn@latest add @dotui/date-picker
    ```

    ## Usage

    Use date pickers to allow users to select a date using a field and a calendar popover.

    ```tsx
    import { Calendar } from "@/components/ui/calendar";
    import { DatePicker, DatePickerContent, DatePickerInput } from "@/components/ui/date-picker";
    import { Label } from "@/components/ui/field";
    ```

    ```tsx
    <DatePicker>
      <Label>Date</Label>
      <DatePickerInput />
      <DatePickerContent>
        <Calendar />
      </DatePickerContent>
    </DatePicker>
    ```

    ## API Reference

    | Prop              | Type                                     | Default     | Description                                          |
    | ----------------- | ---------------------------------------- | ----------- | ---------------------------------------------------- |
    | `value`           | `DateValue`                              | -           | The current value (controlled).                      |
    | `defaultValue`    | `DateValue`                              | -           | The default value (uncontrolled).                    |
    | `minValue`        | `DateValue`                              | -           | The minimum allowed date.                            |
    | `maxValue`        | `DateValue`                              | -           | The maximum allowed date.                            |
    | `granularity`     | `'day' \| 'hour' \| 'minute' \| 'second'`| `'day'`     | The smallest unit that is displayed.                 |
    | `hourCycle`       | `12 \| 24`                               | -           | Whether to use 12 or 24 hour format.                 |
    | `hideTimeZone`    | `boolean`                                | -           | Whether to hide the time zone.                       |
    | `placeholderValue`| `DateValue`                              | -           | A placeholder date that influences the format.       |
    | `isDisabled`      | `boolean`                                | -           | Whether the date picker is disabled.                 |
    | `isReadOnly`      | `boolean`                                | -           | Whether the date picker is read only.                |
    | `isInvalid`       | `boolean`                                | -           | Whether the current value is invalid.                |
    | `isRequired`      | `boolean`                                | -           | Whether the field is required.                       |

    | Event           | Type                          | Description                              |
    | --------------- | ----------------------------- | ---------------------------------------- |
    | `onChange`      | `(value: DateValue) => void`  | Handler that is called when the value changes. |
    | `onOpenChange`  | `(isOpen: boolean) => void`   | Handler that is called when the popover opens or closes. |

    ## Accessibility

    ### Keyboard interactions

    | Key              | Description                                          |
    | ---------------- | ---------------------------------------------------- |
    | `Tab`            | Moves focus between date segments and the calendar button. |
    | `ArrowUp`        | Increments the focused segment value.                |
    | `ArrowDown`      | Decrements the focused segment value.                |
    | `Space` `Enter`  | Opens the calendar popover when the button is focused. |
    | `Escape`         | Closes the calendar popover.                         |

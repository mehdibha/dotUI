## Installation

```npm
npx shadcn@latest add @dotui/input
```

## Usage

Use `Input` to allow users to enter text. Combine with `InputGroup` and `InputGroupAddon` to add icons, buttons, or labels alongside the input.

```tsx
import { Input } from "@/components/ui/input";
```

```tsx
<Input placeholder="Enter text..." />
```

## Sizes

Use the `size` prop to control the input height. Available sizes are `xs`, `sm`, `md` (default), and `lg`.

```tsx
<Input size="xs" />
<Input size="sm" />
<Input size="md" />
<Input size="lg" />
```

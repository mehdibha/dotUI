## Installation

```npm
npx shadcn@latest add @dotui/conversation
```

## Usage

```tsx
import {
  Conversation,
  ConversationContent,
  ConversationScrollButton,
} from '@/components/ui/conversation'
```

```tsx
<Conversation>
  <ConversationContent>
    <Message from="user">
      <MessageBubble>Where should I start?</MessageBubble>
    </Message>
  </ConversationContent>
  <ConversationScrollButton />
</Conversation>
```

`Conversation` keeps the view pinned to the bottom while replies stream in — and
lets go the moment the reader scrolls up, revealing the scroll-to-bottom button.
The edges fade to `--conversation-fade`; override it when the thread sits on a
surface other than the page background.

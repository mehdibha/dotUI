```npm
npx shadcn@latest add @dotui/mention
```

## Usage

Use `Mention` to let users insert tokens — people, channels, anything — by typing
a trigger character while they write. The suggestions list opens at the caret and
filters as they type; selecting an item inserts it inline.

```tsx
import {
  Mention,
  MentionInput,
  MentionItem,
  MentionList,
} from '@/components/ui/mention'
```

```tsx
<Mention>
  <MentionInput label="Comment" />
  <MentionList items={people}>
    {(person) => <MentionItem id={person.id}>{person.name}</MentionItem>}
  </MentionList>
</Mention>
```

The `trigger` prop changes the character that opens the list (`@` by default), and
`getItemText` maps a selected item's key to the text inserted after the trigger.

```tsx
<Mention trigger="#" getItemText={(key) => String(key)}>
  {/* ... */}
</Mention>
```

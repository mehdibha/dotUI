## Installation

```npm
npx shadcn@latest add @dotui/avatar
```

## Anatomy

```tsx
import { Avatar } from "@/components/core/avatar";

<Avatar src=".." alt=".." />;
```

```tsx
import {
  AvatarFallback,
  AvatarImage,
  AvatarPlaceholder,
  AvatarRoot,
} from "@/components/core/avatar";

<AvatarRoot>
  <AvatarImage src=".." alt=".." />
  <AvatarFallback>..</AvatarFallback>
  <AvatarPlaceholder />
</AvatarRoot>;
```

## Options

### Shape

`Avatar` can be either a circle or a square using the `shape` prop.

<Demo name="avatar/demos/shape" />

### Size

Use the `size` prop to control the size of the `Avatar`. The default variant is `md`.

## API Reference

`Avatar` accepts all image attributes.

| Prop       | Type                   | Default    | Description                                                    |
| ---------- | ---------------------- | ---------- | -------------------------------------------------------------- |
| `shape`    | `'circle' \| 'square'` | `'circle'` | The shape of the avatar.                                       |
| `size`     | `'sm' \| 'md' \| 'lg'` | `'md'`     | The size of the avatar.                                        |
| `fallback` | `ReactNode`            | -          | The fallback element to render when an image is not available. |

    ## Installation

    ```npm
    npx shadcn@latest add @dotui/avatar
    ```

    ## Usage

    Use avatars to represent users or entities with an image or initials.

    ```tsx
    import { Avatar } from "@/components/ui/avatar";
    ```

    ```tsx
    <Avatar
      src="https://github.com/mehdibha.png"
      alt="@mehdibha"
      fallback="MB"
    />
    ```

    ## Playground

    ## API Reference

    | Prop       | Type                         | Default    | Description                              |
    | ---------- | ---------------------------- | ---------- | ---------------------------------------- |
    | `src`      | `string`                     | -          | The URL of the avatar image.             |
    | `alt`      | `string`                     | -          | Alternative text for the avatar image.   |
    | `fallback` | `string`                     | -          | The fallback text to display when image fails to load. |
    | `size`     | `'sm' \| 'md' \| 'lg'`       | `'md'`     | The size of the avatar.                  |
    | `shape`    | `'circle' \| 'square'`       | `'circle'` | The shape of the avatar.                 |

    ## Installation

    ```npm
    npx shadcn@latest add @dotui/alert
    ```

    ## Usage

    Use alerts to display important messages that require user attention.

    ```tsx
    import { Alert, AlertDescription, AlertTitle } from "@/components/ui/alert";
    ```

    ```tsx
    <Alert>
      <AlertTitle>Alert Title</AlertTitle>
      <AlertDescription>Alert description</AlertDescription>
    </Alert>
    ```

    ## Playground

    Use the controls below to experiment with different alert props and see the live preview and code update.

    <InteractiveDemo
    component={AlertPlayground} 
    controls={alertControls} 
    fallback={`<Alert variant="neutral">

</Alert>`} />

    ## API Reference

    | Prop      | Type                                                 | Default     | Description                   |
    | --------- | ---------------------------------------------------- | ----------- | ----------------------------- |
    | `variant` | `'default' \| 'success' \| 'warning' \| 'danger' \| 'info'` | `'default'` | The visual style of the alert. |
    | `children`| `ReactNode`                                          | -           | The content of the alert.     |

    ## Installation

    ```npm
    npx shadcn@latest add @dotui/drop-zone
    ```

    ## Usage

    Use drop zones to allow users to drag and drop files or content into a designated area.

    ```tsx
    import { DropZone, DropZoneLabel } from "@/components/ui/drop-zone";
    ```

    ```tsx
    <DropZone>
      <DropZoneLabel>Drop files here</DropZoneLabel>
    </DropZone>
    ```

    ## API Reference

    | Prop              | Type                          | Default | Description                                      |
    | ----------------- | ----------------------------- | ------- | ------------------------------------------------ |
    | `getDropOperation`| `(types: DragTypes, allowedOperations: DropOperation[]) => DropOperation` | - | Handler that returns the drop operation for a drag. |
    | `isDisabled`      | `boolean`                     | -       | Whether the drop zone is disabled.               |
    | `children`        | `ReactNode`                   | -       | The content of the drop zone.                    |

    | Event        | Type                               | Description                                      |
    | ------------ | ---------------------------------- | ------------------------------------------------ |
    | `onDrop`     | `(e: DropEvent) => void`           | Handler that is called when a drop occurs.       |
    | `onDropEnter`| `(e: DropEnterEvent) => void`      | Handler that is called when a drag enters.       |
    | `onDropExit` | `(e: DropExitEvent) => void`       | Handler that is called when a drag exits.        |

    | Data attribute      | Description                                      |
    | ------------------- | ------------------------------------------------ |
    | `[data-drop-target]`| Whether the drop zone is a valid drop target.    |
    | `[data-disabled]`   | Whether the drop zone is disabled.               |

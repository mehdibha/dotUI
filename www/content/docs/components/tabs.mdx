    ## Installation

    ```npm
    npx shadcn@latest add @dotui/tabs
    ```

    ## Usage

    Use tabs to organize content into multiple sections and allow users to navigate between them.

    ```tsx
    import { Tab, TabList, TabPanel, Tabs } from "@/components/ui/tabs";
    ```

    ```tsx
    <Tabs>
      <TabList>
        <Tab id="overview">Overview</Tab>
        <Tab id="usage">Usage</Tab>
        <Tab id="settings">Settings</Tab>
      </TabList>
      <TabPanel id="overview">Overview content</TabPanel>
      <TabPanel id="usage">Usage content</TabPanel>
      <TabPanel id="settings">Settings content</TabPanel>
    </Tabs>
    ```

    ## API Reference

    ### Tabs

    | Prop           | Type                             | Default        | Description                           |
    | -------------- | -------------------------------- | -------------- | ------------------------------------- |
    | `orientation`  | `'horizontal' \| 'vertical'`     | `'horizontal'` | The orientation of the tabs.          |
    | `isDisabled`   | `boolean`                        | -              | Whether the tabs are disabled.        |
    | `selectedKey`  | `Key`                            | -              | The currently selected tab (controlled). |
    | `defaultSelectedKey` | `Key`                      | -              | The default selected tab (uncontrolled). |
    | `children`     | `ReactNode`                      | -              | The tab list and panels.              |

    | Event               | Type                  | Description                              |
    | ------------------- | --------------------- | ---------------------------------------- |
    | `onSelectionChange` | `(key: Key) => void`  | Handler that is called when the selected tab changes. |

    | Data attribute                                | Description                 |
    | --------------------------------------------- | --------------------------- |
    | `[data-orientation="horizontal \| vertical"]` | The orientation of the tabs. |

    ## Accessibility

    ### Keyboard interactions

    | Key                   | Description                                      |
    | --------------------- | ------------------------------------------------ |
    | `Tab`                 | Moves focus to the tab list.                     |
    | `ArrowLeft` `ArrowRight` | Moves focus to adjacent tabs (horizontal).    |
    | `ArrowUp` `ArrowDown` | Moves focus to adjacent tabs (vertical).         |
    | `Enter` `Space`       | Activates the focused tab.                       |
    | `Home`                | Moves focus to the first tab.                    |
    | `End`                 | Moves focus to the last tab.                     |

## Installation

```npm
npx shadcn@latest add @dotui/kbd
```

## Usage

```tsx
import { Kbd } from '@/ui/kbd'
```

```tsx
;<Kbd>⌘</Kbd> + <Kbd>K</Kbd>
```

## Anatomy

Use `Kbd` for a single key. Wrap several in `KbdGroup` to present a key combination as one unit.

```tsx
import { Kbd, KbdGroup } from '@/ui/kbd'
;<KbdGroup>
  <Kbd>⌘</Kbd>
  <Kbd>K</Kbd>
</KbdGroup>
```

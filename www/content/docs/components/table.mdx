    ## Installation

    ```npm
    npx shadcn@latest add @dotui/table
    ```

    ## Usage

    Use tables to display data in rows and columns with support for selection, sorting, and more.

    ```tsx
    import {
      Table,
      TableBody,
      TableCell,
      TableColumn,
      TableHeader,
      TableRow,
    } from "@/components/ui/table";
    ```

    ```tsx
    <Table aria-label="Files">
      <TableHeader>
        <TableColumn isRowHeader>Name</TableColumn>
        <TableColumn>Type</TableColumn>
        <TableColumn>Date Modified</TableColumn>
      </TableHeader>
      <TableBody>
        <TableRow>
          <TableCell>Games</TableCell>
          <TableCell>File folder</TableCell>
          <TableCell>6/7/2020</TableCell>
        </TableRow>
      </TableBody>
    </Table>
    ```

    ## API Reference

    ### Table

    | Prop            | Type                               | Default  | Description                              |
    | --------------- | ---------------------------------- | -------- | ---------------------------------------- |
    | `selectionMode` | `'none' \| 'single' \| 'multiple'` | `'none'` | The type of selection that is allowed.   |
    | `selectionBehavior` | `'toggle' \| 'replace'`        | -        | How multiple selection should behave.    |
    | `selectedKeys`  | `Iterable<Key>`                    | -        | The currently selected keys (controlled). |
    | `defaultSelectedKeys` | `Iterable<Key>`              | -        | The initially selected keys (uncontrolled). |
    | `disabledKeys`  | `Iterable<Key>`                    | -        | The keys that are disabled.              |
    | `sortDescriptor`| `SortDescriptor`                   | -        | The current sort descriptor (controlled). |
    | `children`      | `ReactNode`                        | -        | The table header and body.               |

    | Event               | Type                                      | Description                              |
    | ------------------- | ----------------------------------------- | ---------------------------------------- |
    | `onSelectionChange` | `(keys: Selection) => void`               | Handler that is called when selection changes. |
    | `onSortChange`      | `(descriptor: SortDescriptor) => void`    | Handler that is called when sort changes. |
    | `onRowAction`       | `(key: Key) => void`                      | Handler that is called when a row action is triggered. |

    ## Accessibility

    ### Keyboard interactions

    | Key                   | Description                                      |
    | --------------------- | ------------------------------------------------ |
    | `ArrowUp` `ArrowDown` | Moves focus between rows.                        |
    | `ArrowLeft` `ArrowRight` | Moves focus between cells.                    |
    | `Space`               | Toggles row selection.                           |
    | `Enter`               | Triggers the row action.                         |
    | `Ctrl/Cmd + A`        | Selects all rows (when multiple selection is enabled). |

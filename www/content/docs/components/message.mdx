## Installation

```npm
npx shadcn@latest add @dotui/message
```

## Usage

```tsx
import { Message, MessageAvatar, MessageContent } from '@/components/ui/message'
```

```tsx
<Message align="start">
  <MessageAvatar>
    <Avatar>
      <AvatarFallback>Q</AvatarFallback>
    </Avatar>
  </MessageAvatar>
  <MessageContent>
    <Bubble variant="muted" align="start">
      <BubbleContent>Hey! How can I help you today?</BubbleContent>
    </Bubble>
  </MessageContent>
</Message>
```

`align` sets which side the row sits on and flows down to the content via
context. Compose a `Bubble` inside `MessageContent`, and wrap the avatar around
your own `Avatar`.

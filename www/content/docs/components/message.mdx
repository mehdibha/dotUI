## Installation

```npm
npx shadcn@latest add @dotui/message
```

## Usage

```tsx
import { Message, MessageAvatar, MessageBubble } from '@/components/ui/message'
```

```tsx
<Message from="assistant">
  <MessageAvatar name="dotUI" />
  <MessageBubble>Hey! How can I help you today?</MessageBubble>
</Message>
```

The author set on `Message` flows down to the bubble, avatar, and actions, so
alignment and color stay consistent across a turn. Wrap the header, bubble, and
actions in `MessageBody` when a message needs more than a bubble.

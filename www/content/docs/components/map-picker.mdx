## Installation

```npm
npx shadcn@latest add @dotui/map-picker
```

## Usage

Render `MapPicker` with an initial coordinate. Click anywhere on the map or drag
the marker to pick a location; read the selected `{ lat, lng }` from `onChange`.

```tsx
import { MapPicker } from '@/components/ui/map-picker'
```

```tsx
<MapPicker
  defaultValue={{ lat: 37.7749, lng: -122.4194 }}
  zoom={12}
  onChange={(value) => setLocation(value)}
/>
```

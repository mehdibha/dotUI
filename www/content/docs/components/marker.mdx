## Installation

```npm
npx shadcn@latest add @dotui/marker
```

## Usage

```tsx
import { Marker, MarkerContent, MarkerIcon } from '@/components/ui/marker'
```

```tsx
<Marker role="status">
  <MarkerContent shimmer>Thinking…</MarkerContent>
</Marker>
```

Set `MarkerContent`'s `shimmer` for live streaming text, and `role="status"` so
assistive tech announces it.

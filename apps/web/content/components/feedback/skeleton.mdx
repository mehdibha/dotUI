## API Reference

`Skeleton` accepts all HTML `div` element props and the following:

| Prop   | Type      | Default | Description                                |
| ------ | --------- | ------- | ------------------------------------------ |
| `show` | `boolean` | `true`  | Weather the visual style should be filled. |

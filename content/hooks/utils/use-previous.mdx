## Parameters

| Name       | Type  | Description                                        |
| ---------- | ----- | -------------------------------------------------- |
| `newValue` | `any` | The new value to track and return the previous of. |

## Return value

| Name            | Type  | Description                                    |
| --------------- | ----- | ---------------------------------------------- |
| `previousValue` | `any` | The previous value of the provided `newValue`. |
